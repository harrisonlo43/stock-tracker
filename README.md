# stock-tracker

台股產業趨勢觀察 — 由排程雲端 agent 每營業日晚上自動更新。

## 結構

- `watchlist.txt` — 股票 / ETF 觀察清單（依分類整理）
- `CLAUDE.md` — 給 Claude 的操作說明（手機下指令管理清單用）
- `reports/` — 每次執行產出的產業趨勢報告（Markdown），檔名格式 `YYYY-MM-DD-sector-trend.md`
- `reports/latest.md` — 最新一份報告的複本，方便快速查看

## 手機管理清單

在手機上用 Claude 開一個以本 repo 為來源的雲端 session，直接下中文指令即可：

- 「新增 2454」→ 查名稱與分類，加入清單並自動 commit
- 「移除 00685L」→ 從清單移除
- 「現在清單有哪些」→ 條列目前內容

## 報告內容

搜尋各產業本週整體表現，指出**最強／最弱產業**，用白話說明原因，並更新 Artifact 儀表板趨勢圖。
